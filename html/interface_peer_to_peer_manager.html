- (void) showPeerBrowserWithViewController: (UIViewController *)  viewController
delegate: (id<MultiplayerDelegate>)  delegate 

Shows the MCBrowserVieController and sets the delegate.

This method shows the MCBrowserVieController. With this ViewController the user can find and invite other players. The user which invites is the "server" of the game and should handle all events.

Parameters
viewControllerthe viewController where the MCBrowserViewController should be displayed

- (void) startAdvertisingWithDelegate: (id<MultiplayerDelegate>)  delegate

Starts advertising and sets the delegate.

This method starts the Advertising Assistant (and create it if it is not set yet). Advertising means, that other players can find this device with the MCBrowserViewController (see later)

Parameters
delegateThe delegate which is called, when we get an invite or something else from other connected peers.
